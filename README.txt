AkyFit iPhone website

Upload every file in this folder to GitHub Pages or another HTTPS static host.
Open the HTTPS URL in Safari on your iPhone, then use Share > Add to Home Screen.

Workout planner:
The planner includes Legs A, Legs B, Push A, Push B, Pull A, Pull B,
Hamstring/Calves, and Delts/Arms. The weekly plan defaults to Monday Legs A,
Tuesday Push A, Wednesday off, Thursday Pull A, Friday Hamstring/Calves,
Saturday Delts/Arms, and Sunday off.

Rep ranges and extra notes:
Each exercise includes target rep ranges plus supplemental instructions such as
cluster sets, rest-pause sets, 2 sec contraction pauses, stretch pauses, tempo,
drop sets, and other execution notes. These appear on Today, Log, Planner, and
Workout History. The Log page also includes a Set terminology expander.

Data:
Data is saved on the iPhone using IndexedDB with a localStorage shadow copy.
Use Download backup regularly. Import backup accepts AkyFit/Replit backup JSON.
