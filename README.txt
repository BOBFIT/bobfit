Julius Trainer iPhone website

Upload every file in this folder to GitHub Pages or another HTTPS static host.
Open the HTTPS URL in Safari on your iPhone, then use Share > Add to Home Screen.

Workout planner:
The planner includes Legs A, Legs B, Push A, Push B, Pull A, Pull B,
Hamstring/Calves, and Delts/Arms. The weekly plan defaults to Monday Legs A,
Tuesday Push A, Wednesday off, Thursday Pull A, Friday Hamstring/Calves,
Saturday Delts/Arms, and Sunday off.
Settings includes a Motra Workout Import section for Motra .xlsx exports,
CSV/TSV/text rows, rows copied from Excel/Live Text, and Samsung one-line
pasted workout text. Imported Motra batches can be deleted if they were added
by mistake.

Rep ranges and extra notes:
Each exercise includes target rep ranges plus supplemental instructions such as
cluster sets, rest-pause sets, 2 sec contraction pauses, stretch pauses, tempo,
drop sets, and other execution notes. These appear on Today, Log, Planner, and
Workout History. The Log page also includes a Set terminology expander.

Data:
Data is saved on the iPhone using IndexedDB with a localStorage shadow copy.
Use Download backup regularly. Import backup accepts Julius Trainer, AkyFit, and Replit backup JSON.
If a Samsung browser only offers Camera or Photos when importing, open the backup JSON file,
use Settings > Backup & Restore > Choose backup file and pick it from My Files/Documents.
The paste box is still available as a fallback.

Meals and macros:
Set macro targets in Planner. Today shows progress against those targets,
logged meals, and saved meal suggestions when you are behind. Meals can be
saved from Log, Today, or History and added back to today's log with one tap.
Log combines saved meals and previous logged meals into one Meals Library.
Planner also includes a Meal Plan Builder so saved full-day plans such as
Cutting Day, High Carb Day, or Rest Day can be added to today in one tap.

Dashboard and progress:
Today includes a Daily Score for macros, workout status, peptide reminders,
and weight trend. It also includes a consistency calendar
and coach-style notes from your logged data. Workout Log includes progressive
overload prompts for each exercise, and Log includes Personal Records for best
set, best volume, and estimated 1RM. History includes progress check-ins with
optional photo, weight, waist, mood, and notes.
Each page keeps the first four dropdown sections visible and uses See more tabs
for any extra sections to keep the phone layout cleaner.

Peptide calculator:
The reconstitution calculator is locked to a U-100 1ml syringe rule:
standard peptides use 10 units equals 1mg, with bac water calculated as
vial mg divided by 10. NAD+ and Glow Stack use fixed overrides:
NAD+ 1000mg uses 10ml bac water, and Glow Stack 70mg uses 3ml bac water.
Peptides also includes a Cycle Dashboard showing current cycle week, next dose,
missed doses, total mg used, and remaining days.

Reports:
Settings includes a one-tap branded HTML weekly report export covering workouts,
meals, weight, peptides, adherence, progress check-ins, and a weekly score out
of 100 with a score breakdown.
